## âš™ï¸ System Working
1. A soil moisture sensor continuously measures soil conditions.
2. The system stores current and previous moisture values.
3. If soil moisture drops rapidly, irrigation is triggered **predictively**, even before critical dryness.
4. If soil moisture crosses a predefined dry threshold, irrigation is activated.
5. A rain-aware mode disables irrigation completely during rainfall.
6. Pump ON duration is tracked and converted into approximate water usage.
7. System status and water usage information are displayed on the LCD.
